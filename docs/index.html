KSFP UX Demos

Shareable UX demonstration pages for stakeholders. Enable GitHub Pages and set the source to the docs/ folder.

  • Payment Methods Demo — interactive payment provider tiles
  • Admin Dashboard (public) — full dashboard (may require assets)

How to publish

  1. Push your repo to GitHub.
  2. Go to Settings → Pages and set the source to docs/ folder on the main branch.
  3. After a minute your site will be available at https://<username>.github.io/<repo>/
